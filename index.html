рҹ’і PCE / Core PCE (м—°мӨҖ кіөмӢқ лӘ©н‘ң)
м—°мӨҖ 2% лӘ©н‘ң кё°мӨҖ м§Җн‘ң. Core PCEк°Җ 2.5% мҙҲкіј мң м§Җ мӢң кёҲлҰ¬ мқён•ҳ л…јмқҳ л¶Ҳк°Җ. н•ҳлқҪ нҷ•мқё мӢң н”јлІ— кё°лҢҖ ліёкІ©нҷ”.

рҹ’ј мӢңк°„лӢ№ мһ„кёҲ
мғҒмҠ№ вҶ’ м„ңл№„мҠӨ мқён”Ңл Ҳмқҙм…ҳ кі м°© вҶ’ Core CPI н•ҳл°© кІҪм§Ғ. н•ҳлқҪ вҶ’ м„ңл№„мҠӨ л””мҠӨмқён”Ңл Ҳмқҙм…ҳ вҶ’ м—°мӨҖ лӘ©н‘ң мҲҳл ҙ к°ҖмҶҚ.
